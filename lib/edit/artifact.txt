#
# This file contains information about artifacts.
#

#
# Artifacts 1-21 are amulets, rings, and lights, 22-96 are armors, 97-185 
# are weapons, and 186-190 are diggers.  There are plenty of free
# indexes after each base object kind, so you can add new shields after old.
#
# Artifacts come with descriptions that appear when they are normally 
# identified.  Special thanks to J.R.R Tolkien, without whom the words would 
# be unwritten, the images unconceived.  -LM-
# Thanks also to Charlie Ball, and to Jeff Butler, Neal Hackler, 
# Ethan Sicotte, Pat Tracy, and Yuanli Zhou
#

# Version stamp (required)

V:1.0.2

#
# N: artifact index: artifact name
# I: base object tval (general kind): base object sval (specific type)
# W: depth: rarity: weight in deci-pounds: value
# C: base armor class: damage done when wielded or thrown: 
#    Bonus to Skill: Bonus to Deadliness: Bonus to armor class: 
#    Activation Index (0 if item does not activate)
# P: pval values
# 1, 2, 3: flags associated with up to three pval values
# F: flag | flag | etc
# X: special random attributes
# D: Artifact descriptions.
#
# For more information, see ego_item.txt and /lib/info/modify.txt.
#
